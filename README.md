# AI Extraction and Translation

This project is an AI-powered Python application that automates the extraction of text from documents and translates it into multiple languages.

The system uses **Optical Character Recognition (OCR)** to extract text from images, PDFs, or scanned documents, and integrates **translation APIs** (like Google Translate API) to convert the text accurately.

It is designed to simplify document processing and enable seamless multilingual communication, making it useful for students, professionals, and businesses.

---

## Features
- Extract text from images and PDF files using OCR
- Translate extracted text into multiple languages
- Fast and efficient processing
- User-friendly interface

---

## Technologies Used
- Python
- OCR (Tesseract OCR)
- Translation APIs (Google Translate API)
- Natural Language Processing (NLP)

---

## Use Cases
- Digitizing scanned documents
- Translating documents into different languages
- Extracting information from images for analysis
